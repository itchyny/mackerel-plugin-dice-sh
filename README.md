# mackerel-plugin-dice
Dice plugin for Mackerel.

## Setting
```
[plugin.metrics.dice]
command = "/path/to/mackerel-plugin-dice"
```

## Installation
### Homebrew
```sh
brew install itchyny/tap/mackerel-plugin-dice
```

### Build from source
```bash
go get -u github.com/itchyny/mackerel-plugin-dice
```
